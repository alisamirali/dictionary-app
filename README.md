# Dictionary App

## Dictionary App using Svelte & DictionaryAPI | HTML, CSS, and JavaScript.

- How to run the app:
1. clone the repo
2. run "npm install" into the project folder
3. run "npm run dev"

Live Demo: https://dictionary-app-svelte.vercel.app
